# Lab Activity Box [Front]

## Outline

- Activity Management
- Web Service Connection (e.g. GitHub, Evernode, GoogleCalendar, etc..)
- Weekly Report Submission
- Registered Material Searching

## Components

- Elm - Haskell like Language for Web front development.
